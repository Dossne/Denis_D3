# Tiles

## Версия Unity

- `2022.3.62f2`

## Основные правила игры

- Цель игры — очистить игровое поле, находя и собирая одинаковые плитки (tiles).
- Игра разделена на уровни.
- Уровень представляет собой набор плиток с разными символами (еда, предметы, животные, флаги и другое). Плитки лежат слоями одна на другой, располагаются на уровне стопками.
- Количество плиток на уровне должно быть кратно трём. На каждой плитке только один определённый символ. Количество символов, использующихся на уровне, а также количество плиток и их расположение напрямую влияет на сложность прохождения.
- Можно выбирать только свободные плитки (те, которые не заблокированы сверху другими плитками). Заблокированные плитки становятся доступными для сбора, когда собираются плитки, лежавшие сверху.
- При нажатии на свободную плитку она перемещается в нижнюю панель. В нижней панели есть слоты под семь плиток.
- Нужно собрать 3 одинаковые плитки в панели. Как только на панели собирается тройка одинаковых плиток — все три исчезают.
- Поражение на уровне: все слоты заняты, и ты не можешь собрать тройку, то есть панель внизу заполнена собранными плитками (без одинаковых троек), а на игровом поле остаются плитки.
- Победа на уровне: собраны все плитки на игровом поле, панель внизу пуста (все тройки одинаковых плиток исчезли).
- После прохождения одного уровня запускается следующий, с другим количеством плиток и их расположением.
- Уровни генерируются, исходя из следующих параметров: Количество плиток на уровне; Количество символов, задействованных на уровне; Максимальное количество слоёв плиток; Минимальное количество слоёв плиток; Количество свободных плиток на старте уровня.
- Для первого уровня параметры предопределены заранее: Количество плиток на уровне — 12; Количество символов, задействованных на уровне — 4; Максимальное количество слоёв плиток — 2; Минимальное количество слоёв плиток — 2; Количество свободных плиток на старте уровня — 6.

## Управление

- Игрок нажимает на плитку, чтобы переместить её на панель внизу экрана. Плитка перелетает в нижнюю панель автоматически.
- При победе на уровне в центре экрана появляется кнопка с надписью "Next Level", при тапе по которой запускается следующий уровень.
- При поражении на уровне в центре экрана появляется кнопка с надписью "Try Again", при тапе по которой уровень перезапускается заново.
